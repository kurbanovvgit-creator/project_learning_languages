# Larning languages with IT Tools 🇹🇲

Проект представляет собой набор интеллектуальных инструментов для обработки текста и аудио, созданный с использованием HTML, JS и технологий распознавания речи.

### 🚀 Основные функции:
* [cite_start]**Audio to Text:** Конвертация аудиофайлов в текстовый формат.
* [cite_start]**Speech Recognition:** Живое распознавание речи через микрофон[cite: 119].
* [cite_start]**Text Checker:** Инструмент для анализа и проверки текста[cite: 119].
* [cite_start]**Information Portals:** Разделы с новостями (habarlar) и правительственной информацией (hokumet)[cite: 116, 117].

### 🛠 Технологии:
- HTML5 / CSS3
- JavaScript (Web Speech API)
- UI/UX дизайн в национальном стиле
